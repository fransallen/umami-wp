Easily add Umami Analytics to your WordPress.

== Description ==

Umami is a simple, fast, website analytics alternative to Google Analytics. This plugin allows you to add Umami to your website easily.

== Installation ==

#### INSTALL STATICALLY FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select "Add New";
2. Search for "Umami Analytics";
3. Activate Umami Analytics from your Plugins page;
4. Access the plugin settings from the "Settings > Umami Analytics" menu.

#### INSTALL STATICALLY MANUALLY

1. Upload the "umami" folder to the /wp-content/plugins/ directory;
2. Activate the Umami Analytics plugin through the "Plugins" menu in WordPress;
3. Access the plugin settings from the "Settings > Umami Analytics" menu.
